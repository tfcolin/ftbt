# ftbt

#### 介绍

一款战棋游戏引擎及基于 ncurses 的 UI, 采用 go 语言开发. 支持调整规则, 设计关卡, 和对接不同类型的 UI

#### 软件文档

详见 doc/ftbt.pdf


#### 安装教程

git clone https://gitee.com/tfcolin/ftbt
cd ftbt
go build .
cd ftbt_nc
go build .
cd ../ftbt_ai_nc
go build .
cd ..
